Philosophy of this website

This website is meant to be simple. It is using only HTML and CSS. No PHP, JS or any frameworks. It is intended for me to be fun to create and edit this website manually, to force me to make it as simple as possible. Because I think there is strength in simplicity.

That also means there will be no footer, header or nav bar with navigation. As it would be a mess to manually edit it on each site. Instead each site will have links that will move you further.

It is also using the Resurrect 64 Palette by Kerrie Lake from LOSPEC. Because I really love it. And supports light and dark mode based on your system settings.
