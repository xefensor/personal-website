Philosophy of this website

This website is meant to be simple. It is using only HTML and CSS. No PHP, JS or any frameworks. It is intended for me to be fun to create and edit this website manually, to force me to make it as simple as possible. Because I think there is strength in simplicity.

It is also using the Resurrect 64 Palette by Kerrie Lake from LOSPEC. Because I really love that color palette.
