Useful browser extensions

Extensions for browsers I find useful and use on a daily basis.

uBlock Origin

Most people know uBlock as the best adblock. But it is much more than that. Besides ads it also blocks trackers, coin miners, popups, malicious websites and much more, all of which you can customize.

A must have extension for web security.

I still don't care about cookies

An extension that will hide all cookie warnings from websites, so that you can surf the web without irritations. It will try to hide the warning or accept it, when it is needed for proper functioning of the website.

It's also an open-source fork of "I don't care about cookies", which was acquired by Avast.

Dark Reader

Useful extension for people like me who need websites to render in a dark mode, even if they don't support it. Dark reader can detect if a website doesn't support a dark mode and than changes it, so that you don't even realize it was changed. It can also apply changes, only when you have your browser in dark mode, which makes it a perfect extension for users, who change their system theme.

Be warned this extension can be resource heavy on some websites, where it doesn't function properly. And it also sometimes wrongly detects, that the website is not using a dark mode, even if it does. For that reason, I am using it in a manual mode and I am adding the websites, it should work on, myself.

Bitwarden

Bitwarden of course isn't mainly an extension, it is a proper password manager. But their browser extension functions very well and I use it a lot, so I am including it here.

Tampermonkey

This plugin allows you to download, create and use small scripts that can allow you to add additional features to websites. For example I use a script called Simple YouTube Age Restriction Bypass, which as the name says bypasses age restricted videos on YouTube.

SponsorBlock

Do you like watching content on YouTube and do you like not wasting your time? Then this extension is for you! It will automatically skip over any parts of the video with sponsorship and if you set it up, it can also skip over intros, outros, reviews/recaps and much more.

All of this functions only by the help of community, which registers segments of the videos, so that the plugin can skip it.

DeArrow

By the same person as SponsorBlock. This extension replaces clickbait titles and thumbnails with proper version of them, also submitted by the community.

Together with SponsorBlock a must have extension for browsing YouTube.

Plasma Integration

Very useful for users of the KDE Plama desktop. It allows the browser to communicate with it, so that you can better control media, send links via KDE Connect, show and control downloads in Plasma's notifications, find browser tabs and more.

Augmented Steam

Improves the Steam experience by a lot. Adds price info and regional pricing, store page customization and useful links, highlights your wishlisted or owned games, adds additional filters for better browsing, advanced market features and much more. All of which you can customize via it's many options.
